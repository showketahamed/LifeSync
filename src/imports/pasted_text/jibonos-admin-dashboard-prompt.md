# Figma Prompt – JibonOS Bangladesh Enterprise Admin Dashboard

Create a complete enterprise-grade SaaS Admin Dashboard UI/UX for a Bangladesh-focused personal life management platform named **JibonOS Bangladesh**.

The admin dashboard should look modern, premium, scalable, and comparable to products like Stripe Dashboard, Vercel, Notion, Linear, Supabase, and Clerk.

The design must be responsive, clean, component-based, developer-friendly, and ready for implementation using Next.js, TypeScript, Tailwind CSS, Shadcn UI, Prisma, PostgreSQL, and Auth.js.
